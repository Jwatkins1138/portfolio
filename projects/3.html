private events

Private Events is a web application made with Ruby on Rails and currently deployed via Heroku. The application allows for users to create a profile, create events and 'attend' events created by themselves and other users.

A user can view all upcoming events all past events or view events based on who created them, who is attending or has attended.

additional gems and database

The Devise gem is utilized for user/session authentication, Bulma SASS framework is used for the styling, Gravatar is used for the user profile images and Simple Form is used for the forms.

The postgresql database has tables for Users, Events and Invitations.
