www.allcharts.site

This is a React SPA that pulls financial data from the Alpha Vantage API and makes charts with said data using Chart.js.

There is also a dedicated Rails backend that provides user functions. Sessions and authentication are done through Devise and Devise-JWT.

additional info

Users can currate lists of watched stock tickers. Users can also comment on any stock ticker they like, anyone can view the comments.

Users can choose various time frames of chart data, they can also view general information of any selected ticker symbol.
