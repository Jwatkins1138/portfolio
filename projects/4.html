dmitter

Dmitter is a twitter style message board application where users can post and see a feed of 'dmeets' from themselves and other users whose identities are hidden unless the user is logged in.

Dmitter is named after my cat Dmitri and I use it to post wordle scores.

additional gems and database

The Devise gem is utilized for user/session authentication, Bulma SASS framework is used for the styling, Gravatar is used for the user profile images and Simple Form is used for the forms.

The postgresql database contains tables for users and dmeets.
