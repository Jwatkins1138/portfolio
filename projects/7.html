This is fake web shop built with React.

A user can browse items in the shop sorted by type or price.

Items in the cart will show up in the mini-cart side bar and then can be viewed in more detail in the actual cart.

additional info

This app utilises react browser router dom for navigation, the components are all functional and utilize useEffect and useState for dynamic updates.

The item images were generated by entering the nonsensical descriptions into the AI image generator at https://www.craiyon.com/.

There is no backend, however I faked a loading screen for fun.
