wheres the guy

This is a Where's Waldo game comprised of a React frontend connected to a Rails API backend.

A user can select a level, once the level is selected a timer starts and they must locate each of the five hidden characters within the image.

Once all the characters have been located the timer stops and the user is prompted to enter their name for the leaderboard.

additional info

The backend is a simple rails API which stores location data for the characters and the scores of players who have completed a game.

The backend repo can be viewed here: https://github.com/Jwatkins1138/wheres-waldo-api
