flickr-feed

This is a rails app that is connected to the Flickr API in which a user can search for a someone's Flickr ID and see a feed of thier public photos.

The feed defaults to my own Flickr public stream and shows pages of nine photos at a time.

additional gems and database

This app has no database and operates via a static page controller searching and displaying photos via the Flickr API gem.
