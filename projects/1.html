facelord

Facelord is a facebook style social media application built with Ruby on Rails.

A user can send a friend request to another user who can accept or deny the request at which point they will become friends (or not) and be able to view eachothers posts and profiles.

In addition to making posts, users can also "like" a post or add a comment to it.

additional gems and database

The Devise gem is utilized for user/session authentication as well as Omniauth through the facebook API allowing users to login via thier facebook credentials, Bulma SASS framework is used for the styling. The current version deployed to Heroku uses Gravatar for the user profile images but I am planning to add an image upload function through active storage and deploy to AWS. Simple Form is used for the forms.

The PostgreSQL database has tables for users with a bi-direction friend relatioship through a friendship model, posts, comments and likes.
